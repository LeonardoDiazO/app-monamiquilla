# Mon Ami Quilla 🎨 – Decoraciones Artesanales

¡Bienvenido al repositorio de **Mon Ami Quilla**! Una tienda virtual dedicada a la creación de decoraciones artesanales únicas en Barranquilla, Colombia. 🇨🇴

> **"Trae tu idea, nosotros los pinceles y colores."**

![Mon Ami Logo](img/Logo_monami-modified.png)

## 📌 Sobre el Proyecto

Mon Ami es un emprendimiento que transforma materiales simples en piezas llenas de vida y color para eventos especiales. Este repositorio contiene el código fuente de nuestra plataforma web, diseñada para ofrecer una experiencia de compra fluida y personalizada.

## ✨ Características Principales

- **Catálogo Interactivo**: Explora nuestros productos destacados y filtrados por categorías.
- **Carrito de Compras**: Gestiona tus productos favoritos de forma local antes de realizar el pedido.
- **Integración con WhatsApp**: Realiza pedidos y solicita cotizaciones personalizadas directamente a través de un chat directo.
- **Diseño Responsivo**: Optimizado para dispositivos móviles y escritorio, con una estética moderna y colorida.
- **Efectos Visuales**: Micro-animaciones y transiciones suaves para una mejor experiencia de usuario.

## 🛠️ Materiales con los que Trabajamos

Dividimos nuestras creaciones en tres categorías principales basadas en el material:

1.  **Icopor ☀️**: Letras 3D, coronas, centros de mesa y figuras decorativas pintadas a mano.
2.  **Foamy 🎨**: Flores, kits de Baby Shower, guirnaldas y figuras temáticas duraderas.
3.  **Cartón 🎈**: Cajas decoradas, marcos porta retratos y letreros personalizados.

## 🚀 Tecnologías Utilizadas

- **Estructura**: HTML5 semántico.
- **Estilos**: CSS3 (Vanilla) con variables para una gestión ágil del diseño.
- **Lógica**: JavaScript (Modular / ES Modules) para la gestión del carrito y productos.
- **Iconografía**: [Font Awesome 6](https://fontawesome.com/).
- **Tipografía**: Google Fonts (Inter, Roboto, etc.).

## 📂 Estructura del Proyecto

```text
App_MonamiQuilla/
├── css/
│   ├── main.css      # Estilos globales y componentes
│   └── index.css     # Estilos específicos (si aplica)
├── js/
│   ├── main.js       # Lógica de navegación y UI
│   ├── products.js   # Base de datos local de productos
│   └── cart.js       # Lógica del carrito de compras
├── img/              # Recursos visuales (Logos, imágenes)
├── index.html        # Página principal
└── shop.html         # Catálogo de productos
```

## 💻 Cómo Ejecutar el Proyecto

1.  Clona este repositorio:
    ```bash
    git clone https://github.com/LeonardoDiazO/App_MonamiQuilla.git
    ```
2.  Abre el archivo `index.html` en tu navegador preferido. No requiere de servidores complejos, ya que es una aplicación Front-end pura.
